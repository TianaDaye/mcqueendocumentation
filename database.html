Database Documentation

Hosting

The database for the GyroCar reservation consists of a single MySQL database hosted on AWS. The database is available in 2 AWS availability zones to ensure high availability. It is currently setup to have daily snapshots and weekly backups. The database is not publicly accessible and can only be accessed through the backend subnet in AWS. The database resources can be scaled vertically as usage of the site grows.

Schema

The database schema is fully normalized to third normal form to limit data redundancy. The full ER diagram can be found here>. The schema was developed in MySQL workbench, and the MWB file has been included so that the schema can easily be adjusted as needed.
